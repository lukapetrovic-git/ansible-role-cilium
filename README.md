# ansible-role-cilium
Ansible role to install/upgrade/downgrade/change Cillium (config) on k8s cluster.

# Requirements
- Ansible 2.10 or later

# Tested on
- Ubuntu 20.04 LTS
- Ubuntu 22.04 LTS

# Role Variables
All settable variables with explanations and links are located in the defaults/main.yml
